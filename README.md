# 💳 Credit Scoring Analysis

## 📝 Project Overview
This Machine Learning project aims to evaluate customer creditworthiness, predicting whether a credit or loan application should be accepted or rejected. The model includes advanced interpretability features to explain the algorithm's decisions.

## 🎯 Objectives & Results
* **Predictive Modeling:** Trained a Random Forest Classifier on a dataset containing over 338,000 records.
* **Evaluation Metrics:** Analyzed performance using Accuracy, ROC-AUC, and Confusion Matrix.
* **Explainable AI (XAI):** Utilized the SHAP (SHapley Additive exPlanations) library to generate *Summary Plots* (global feature importance) and *Waterfall Plots* (local explanation for individual customer decisions).

## 🛠️ Technologies & Libraries
* Python
* Pandas, NumPy, Matplotlib, Seaborn
* Scikit-Learn
* SHAP
